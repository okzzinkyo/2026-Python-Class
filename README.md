## 🛠 사용 환경

- **언어:** Python 3.14
- **에디터:** Cursor (VS Code 기반)
- **환경:** 가상 환경 (.venv) 사용 중

## 🚀 시작하는 방법

이 프로젝트를 실행하려면 다음 단계를 따라주세요.

1. 가상 환경 설정 및 활성화:
  ```bash
    python3 -m venv .venv
    source .venv/bin/activate
  ```
2. 필요한 라이브러리 설치:
  ```bash
    pip install -r requirements.txt
  ```

## 📦 의존성 관리
라이브러리를 새로 설치한 후에는 다음 명령어로 `requirements.txt`를 업데이트해 주세요:
  ```bash
    pip freeze > requirements.txt
  ```
