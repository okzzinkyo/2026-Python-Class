# [강의] 파이썬프로그래밍1 실습

## 🛠 사용 기술
- Python 3.14
- gTTS (음성 변환)
- Pillow (이미지 처리)

## 🚀 시작하는 방법
이 프로젝트를 실행하려면 다음 단계를 따라주세요.

1. 가상 환경 설정 및 활성화:
    ```bash
    python3 -m venv .venv
    source .venv/bin/activate

2. 필요한 라이브러리 설치:
    ```bash
    pip install -r requirements.txt

3. 라이브러리 목록 업데이트:
    ```bash
    pip freeze > requirements.txt
