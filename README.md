# [강의] 박소현 교수님 - 파이썬 프로그래밍1 실습

## 📝 강의 정보

- **강의명:** 파이썬 프로그래밍1
- **담당 교수님:** 박소현 교수님
- **학기:** 2026학년도 1학기

## 📂 실습 폴더 구조

각 주차별로 폴더를 나눠서 관리하고 있습니다.

- `01/` : 파이썬 소개 + 기초 문법, 거북이 그래픽, gTTS, pillow활용 실습
- `02/` : 수식과 계산 - 단답형 문제 채점 프로그램, 산수 퀴즈, 영어 약어 추출하기, 태양빛이 지구에 도달하는 시간 구하기 실습
- `03/` : if ... else 구문 - 초과 수당 임금 계산, 초과 수화물 수수료 구하기, 팁 계산하기, 이차 방정식 해 구하기, 숫자 맞추기 게임 등 실습
- `04/` : 반복문(for, while) - range 활용, 터틀로 도형 반복 그리기, 입력 받은 성적 평균 계산하기 등

## 🛠 사용 환경

- **언어:** Python 3.14
- **에디터:** Cursor (VS Code 기반)
- **환경:** 가상 환경 (.venv) 사용 중

## 🚀 시작하는 방법

이 프로젝트를 실행하려면 다음 단계를 따라주세요.

1. 가상 환경 설정 및 활성화:
  ```bash
    python3 -m venv .venv
    source .venv/bin/activate
  ```
2. 필요한 라이브러리 설치:
  ```bash
    pip install -r requirements.txt
  ```
